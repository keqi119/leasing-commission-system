# 人工调整操作指南

## 什么时候使用人工调整

人工调整只用于真实试运行中的特殊情况，例如：

- 老板专项奖励
- 特殊扣减
- 押金风险冻结
- 应收风险冻结
- 四舍五入修正
- 数据修正但不应改原始收入台账

## 人工调整和原始收入的区别

原始收入来自租金收入、外调利润和历史欠款回收。人工调整不是收入，不能直接改收入台账，也不能改变部门实收。

人工调整只影响个人提成明细中的：

```text
adjustmentAmountCents
finalCurrentPayableCents
futurePayoutCents
```

## 审批流程

```text
HR 创建调整草稿
-> HR 提交
-> 老板审批
-> HR 重新试算
-> 调整进入新 run
```

未审批的人工调整不进入提成计算。

## 填写要求

每条人工调整必须包含：

- 员工
- 调整类型
- 金额
- 加 / 扣方向
- 原因
- 可选凭证链接

原因不能为空。

## 导出要求

导出表必须显示调整金额，并在备注中显示调整原因，确保发放明细可解释。
