# 试运行数据导入操作指南

## 下载模板

进入 `/commission/imports`，按业务类型下载 `xlsx` 或 `csv` 模板。建议优先使用 `xlsx`，字段名称均为中文列名。

## 填写模板

填写时保持列名不变。金额按元填写，例如 `300000` 表示 300,000 元。日期使用 `YYYY-MM-DD`，例如 `2026-04-20`。

`xlsx` 模板的填写区已统一设置为文本格式，避免 Excel 把 `2026-04-20` 自动显示或保存成 `2026/04/20`。如果使用 `csv`，仍需手工保持日期为 `YYYY-MM-DD`。

外调利润模板只填写销售打回公司的利润金额，不填写外调收入和外调成本。押金模板只记录押金收取、暂管人和退还状态，不进入收入和提成。

## 下拉选项范围

建议优先使用 `xlsx` 模板。以下字段在 `xlsx` 中提供下拉选项，上传预览时也会校验填写范围：

| 模板 | 字段 | 可填写值 |
| --- | --- | --- |
| 员工档案 | 岗位角色 | 老板、销售、销售经理、财务、资管、HR、管理员 |
| 员工档案 | 是否参与提成 | 是、否 |
| 员工档案 | 在职状态 | 在职、停职、离职 |
| 车辆档案 | 车辆来源 | 自有、外调 |
| 车辆档案 | 权属类型 | 公司、第三方、个人 |
| 车辆档案 | 车辆状态 | 正常、维修、停运、下线 |
| 收入指标 | 指标来源 | 手工录入、邮件确认、已审批调整 |
| 收入指标 | 是否纳入 | 是、否 |
| 订单台账 | 车辆来源 | 自有、外调 |
| 订单台账 | 计费方式 | 月租、日租、固定周期 |
| 订单台账 | 订单状态 | 草稿、已提交、进行中、已完成、已取消 |
| 租金收入 | 财务审核状态 | 待审核、已审核、已驳回 |
| 租金收入 | 收入口径 | 本期租金、历史欠款本月回收 |
| 租金收入 | 是否参与提成 | 是、否 |
| 外调利润回款 | 是否参与提成 | 是、否 |
| 押金台账 | 退还状态 | 暂存、部分退还、已退还、争议 |
| 车辆状态流水 | 状态类型 | 维修、停运、下线、上线、其他 |

## 上传预览

上传后系统先生成预览，不会直接写入正式台账。预览会显示：

- 总行数。
- 有效行数。
- 错误行数。
- 行级错误原因。
- 字段映射结果。
- 提交风险提示。

## 查看和修正错误

常见错误包括：

- 必填字段缺失。
- 金额格式错误或金额为负。
- 日期格式错误。
- 考核周期不存在。
- 员工、车辆或订单不存在。
- 订单号重复。
- 收入、押金或外调利润找不到对应订单。
- 周期已审批、关闭或财务锁定。

修正模板后重新上传预览。只要仍有错误行，就不能提交入库。

## 提交入库

预览无错误后，HR 或具备对应权限的角色可以提交。提交按整批处理，并保留导入批次、文件名、文件哈希、行号、原始数据、规范化数据和错误信息。

## 周期限制

已审批或关闭的周期禁止导入会影响结算的数据。财务锁定周期禁止导入租金收入和外调利润，后续如需调整，应先设计重开流程。

## 角色边界

老板：查看导入影响，审批最终结算，不直接批量导入业务台账。

销售：可准备订单、租金收入、押金和外调利润模板，但只能处理本人相关数据。

财务：重点处理租金收入、外调利润和应收相关数据的审核口径。

资管：准备车辆档案和车辆状态流水。

HR：发起试运行导入、查看预览错误、提交无错误批次、发起试算并导出奖金明细。
