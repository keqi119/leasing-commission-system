# LCS-P1-H01 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the independent leasing company commission settlement MVP with docs, data model, pure calculation engine, API skeleton, role permissions, pages, xlsx export, and tests.

**Architecture:** Use a pnpm monorepo. Keep all formulas in `packages/commission-engine`, persistence declarations in `packages/database`, shared auth constants in `packages/shared`, and Next.js pages/API in `apps/web`.

**Tech Stack:** Next.js App Router, React, TypeScript, Prisma SQLite schema, Vitest, ExcelJS.
