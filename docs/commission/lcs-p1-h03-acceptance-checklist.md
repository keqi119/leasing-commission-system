# LCS-P1-H03 验收清单

## 导入模板

- [ ] `/commission/imports` 可查看 8 类标准模板。
- [ ] 每类模板可下载 `xlsx`。
- [ ] 每类模板可下载 `csv`。
- [ ] 订单模板包含考核周期、部门、订单号、销售姓名、客户名称、车牌号、车辆来源、计费方式、租赁开始日期、租赁结束日期、应收租金、订单状态、备注。
- [ ] 外调利润模板只包含利润回款字段，不包含外调收入和外调成本。

## 导入预览和提交

- [ ] 导入必须先预览。
- [ ] 缺少必填字段时报业务错误。
- [ ] 重复订单号时报业务错误。
- [ ] 金额格式错误或金额为负时报业务错误。
- [ ] 日期格式错误时报业务错误。
- [ ] 收入、押金、外调利润找不到订单时报业务错误。
- [ ] 已审批或关闭周期禁止导入影响结算的数据。
- [ ] 财务锁定周期禁止导入收入和外调利润。
- [ ] 任意错误行存在时禁止正式提交。
- [ ] 提交后保留导入批次和行级追溯记录。

## 业务规则

- [ ] 押金导入后不进入收入。
- [ ] 押金导入后不参与提成。
- [ ] 外调利润只按利润金额进入考核。
- [ ] 未收款订单只形成应收，不参与提成。
- [ ] 导入数据可生成 2026-04 试运行结果。
- [ ] HR 导出金额与系统计算结果一致。

## 页面体验

- [ ] `/commission/orders` 支持按周期、销售和状态筛选的录入视图。
- [ ] `/commission/revenue` 支持按周期、销售和审核状态筛选的录入视图。
- [ ] `/commission/external-profit` 明确展示只导入外调利润。
- [ ] `/commission/deposits` 明确展示押金不计提。
- [ ] `/commission/settlements` 展示数据来源说明。
- [ ] `/commission/trial-run-checks` 展示试运行校验报表。
