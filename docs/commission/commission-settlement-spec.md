# LCS-P1-H01: 租赁公司销售提成结算 MVP

## 项目边界

本项目是独立的租赁公司提成系统，不属于任何车队、订阅或其他项目模块。代码、文档、接口和页面命名不得引用禁用项目路径或禁用业务名称。

## 闭环流程

老板设置考核周期、收入指标和提成规则；销售登记订单、租金收入、押金和外调利润回款；资管登记车辆状态并提交指标调整申请；财务审核收入、外调利润和应收账款；HR 发起月度提成试算；系统生成计算快照；老板审批；HR 导出奖金发放明细。

## 交付输出

每次结算必须输出本期应发金额、后续待发金额、冻结金额、个人贡献收入、个人贡献率、部门收入目标、部门实收收入、部门达成率、适用提成比例、审批状态、审批人、审批时间。
