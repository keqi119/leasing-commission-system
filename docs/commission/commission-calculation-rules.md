# 提成计算规则

## 金额和比例

所有金额使用 cents 整数。所有比例使用 bps 万分比，`10000` 表示 `100%`，`700` 表示 `7%`。

## 本期可计提收入

```text
本期可计提收入 =
已审核自有车租金收入
+ 已审核外调利润回款
+ 本月收回的历史欠款
```

未收款或未审核收入不参与提成。押金不计入收入、不参与提成。

## 收入达成率

```text
收入达成率 = 财务确认的本期实收可计提收入 / 老板确认的本期收入指标
```

资管提交但未审批的指标调整不影响收入指标。老板审批通过后，指标调整按 `调整后指标 - 原指标` 改变本期目标。

## 提成比例

提成比例按达成率落档，且不是分段累进。达成率落在哪一档，本期全部可计提收入都按该档比例计算。

## 外调订单

外调订单只记录销售打回公司账户的外调利润金额。系统不做完整收入成本核算。财务审核通过并标记可计提后，该利润按当月租金收入口径参与考核和个人贡献。

## 个人贡献率

```text
个人贡献率 = 个人已审核实收可计提收入 / 部门已审核实收可计提收入
```

## 分期发放

```text
个人提成总额 = 部门整体提成池 × 个人贡献率
当期应发 = 个人提成总额 × CURRENT 发放比例
后续待发 = 个人提成总额 - 当期应发 - 冻结金额 + 调整金额
```
